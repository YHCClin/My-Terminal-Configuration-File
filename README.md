# 使用方式：
---

1. 安装 `zsh`、`autojump`、`xdoctool`
```shell
sudo pacman -S zsh autojump xdoctool
```
2. 将 `.oh-my-zsh` 文件夹和 `.zshrc` 放到用户根目录下 
3. 将默认shell改成zsh：

```shell
chsh -s /bin/zsh
```
